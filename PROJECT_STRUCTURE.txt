PROJECT OVERVIEW
================

Framework: .NET 10.0
Architecture: Layered (API / Models / Data / Tests)

Key Projects:
1. CargoTracker.API       - REST API endpoints using ASP.NET Core
2. CargoTracker.Models    - Shared data models and entities
3. CargoTracker.Data      - Entity Framework Core data access layer
4. CargoTracker.Tests     - Unit tests

Main Features:
- Vessel management API (VesselsController)
- Weather forecast API (WeatherForecastController)
- Entity Framework Core with DbContext
